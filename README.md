Sistema Bancario - Arquitectura de Tres Capas

Aplicación de ejemplo que simula la gestión básica de un sistema bancario.
El proyecto está desarrollado en C# utilizando Windows Forms y SQL Server bajo el modelo de arquitectura de tres capas.

Tecnologías utilizadas

C#

Visual Studio

.NET Framework

SQL Server

Windows Forms

Arquitectura del sistema

El proyecto está organizado en tres capas:

Capa de Presentación
Contiene los formularios y la interfaz gráfica que permite la interacción con el usuario.

Capa de Negocio
Contiene la lógica del sistema y las reglas de negocio.

Capa de Datos
Se encarga de la conexión con la base de datos y la ejecución de consultas SQL.
